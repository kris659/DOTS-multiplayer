## Description

This project is a simple multiplayer prototype built with Unity ECS (DOTS) and Netcode for Entities.
This was my first attempt using ECS and Netcode for Entities, and I learned a lot during the process.
Overall, I had a very positive experience with ECS and I find the approach interesting and promising.

## Credits
The project is based on official Unity samples.
